Food Delivery Data Analysis with Pandas

Project Overview

This project performs an exploratory and descriptive analysis of a Food Delivery dataset using Python, Pandas, and NumPy. The goal is to understand customer ordering patterns, restaurant performance, delivery efficiency, and customer ratings.

Objectives

* Understand the structure of the dataset
* Perform data exploration and descriptive statistics
* Identify the most popular cuisines
* Analyze customer ordering behavior
* Examine delivery and food preparation times
* Evaluate restaurant performance based on order volume
* Generate insights from the dataset

Tools Used

* Python
* Pandas
* NumPy
* Jupyter Notebook

Dataset Features

The dataset contains information such as:

* Order ID
* Customer ID
* Restaurant Name
* Cuisine Type
* Cost of Order
* Day of the Week
* Customer Rating
* Food Preparation Time
* Delivery Time

Analysis Performed

Data Exploration

* Dataset shape
* Column inspection
* Data types
* Missing value analysis

Descriptive Analysis

* Summary statistics
* Average order cost
* Average delivery time
* Average preparation time

Business Analysis

* Most ordered cuisine types
* Orders by day of the week
* Top restaurants by number of orders
* Customer rating distribution
* Average cost and delivery time by day

Key Findings

* Weekend orders were higher than weekday orders.
* American cuisine was the most frequently ordered cuisine.
* Most customers provided positive ratings.
* Some restaurants received significantly more orders than others.
* Delivery and preparation times were consistent across most orders.

Conclusion

The analysis shows that customer demand is higher during weekends, with certain cuisines and restaurants dominating order volume. Understanding these trends can help food delivery businesses improve operations, customer satisfaction, and overall performance.
